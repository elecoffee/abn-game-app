# ABN Reactor: 教育用デジタルカードゲーム仕様書

## 1. プロジェクト概要
本アプリは、酸塩基中和反応の化学量論（係数関係）を、ゲームプレイを通じて直感的に学習することを目的としたデジタルカードゲームです。添付資料「Acid-Base Neutralization（ABN）ゲーム 説明書」のルールを忠実に再現しつつ、デジタルならではの視覚効果と自動判定機能を備えています。

## 2. ゲームシステム
### 2.1 基本ロジック
- **中和反応の再現**: プレイヤーは手札の「酸」と「塩基」を組み合わせ、場にある「塩／水」カードの条件を満たすことでカードを獲得します。
- **化学量論の自動判定**: 選択されたカードの組み合わせが、正しい化学反応式に基づいているかをシステムが瞬時に判定します。
- **勝利条件**: 獲得したカードに記載された「水分子（H₂O）の数」がスコアとなり、最終的な合計点が高いプレイヤーが勝者となります。

### 2.2 特殊効果の実装
- **逆合成 (Reverse)**: プレイ順が逆回転になります。
- **ワンポット合成 (Skip)**: 次のプレイヤーのターンをスキップします。
- **濃度調整 (Multiplier)**: 「x2 mol」「x3 mol」カードを使用することで、酸カード1枚を複数枚分としてカウントできます。

## 3. デザインコンセプト
### 「ラボラトリー・ネオン・フューチャー」
- **視覚的対比**: 酸をピンク、塩基をシアンのネオンカラーで表現。これはリトマス試験紙の色の変化を現代的に解釈したものです。
- **没入感**: 近未来の化学研究所をイメージした背景と、動的なパーティクル演出により、学習を「実験」というエキサイティングな体験に昇華させます。

## 4. 技術スタック

- **Styling**: Tailwind CSS 4, shadcn/ui
- **Animation**: Framer Motion

## 5. 教育的設計意図
- **係数への注目**: 単に物質を合わせるだけでなく、「H₂Oがいくつ出るか」をスコアに直結させることで、化学反応式の係数（量論関係）への意識を自然に高めます。
- **多価の酸・塩基の理解**: 硫酸（2価）やリン酸（3価）など、価数が異なる物質の組み合わせをパズル的に解くことで、複雑な中和反応の理解を助けます。
- **即時フィードバック**: デジタル化により、誤った組み合わせを即座に指摘・防止し、正しい知識の定着を促します。
